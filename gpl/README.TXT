Run xx-build-all.sh script to build all GNU toolchain and cross the
fingers. If you have problems due to very old gnu assembler or gcc
please uncomment the lines relative to 09-build-binutils.sh and
10-build-gcc.sh inside xx-build-all.sh

_download.sh is used to download all sources files (wget is required).

_create-scripts.sh is used to archive a new version of build scripts.
